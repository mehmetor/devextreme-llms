# DevExtreme LLM Hub

This project generates AI-optimized documentation for DevExtreme components, designed to be easily consumed by AI assistants like Cursor, Claude, and Windsurf.

## üìã Prerequisites

This tool requires the **DevExtreme source code** to generate documentation.
1. Clone the [DevExtreme GitHub](https://github.com/DevExpress/DevExtreme) repository to your machine.
2. Ensure the repo is available in the root of this project as `DevExtreme/` or as a symbolic link (symlink).

## üöÄ Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Link Packages (Optional)
To use local DevExtreme source code, you can link the packages:
```bash
# devextreme
cd DevExtreme/packages/devextreme/artifacts/npm/devextreme && npm link

# devextreme-react
cd DevExtreme/packages/devextreme-react/npm && npm link
```

### 3. Generate Documentation
To generate documentation for a single component:
```bash
node generate-doc.js <ComponentName>

# Example:
node generate-doc.js DataGrid
```

### 4. Generate All Documentation
To automatically scan and generate documentation for all DevExtreme components:
```bash
node generate-all-docs.js
```

## ü§ñ AI Optimization

The system uses a three-layer structure to help AI assistants understand components effectively:

1.  **Main `llms.txt`**: A central map of all available platforms (React, Angular, etc.) and components.
2.  **Component-Specific `llms.txt`**: (e.g., `react/DataGrid/llms.txt`) A summary and directory map for a specific component.
3.  **Component-Specific `llms-full.txt`**: (e.g., `react/DataGrid/llms-full.txt`) **[Most Important]** The entire API documentation (interfaces, types, variables) merged into a single file. Providing this file to your AI assistant allows it to fully understand the component's API.

## üìÅ Directory Structure

- `DevExtreme/`: DevExtreme monorepo (source code).
- `react/`: Generated React Markdown documentation and AI files.
- `docs-react-final/`: General API dump generated by TypeDoc.
- `generate-doc.js`: Main automation script.
- `llms.txt`: Central AI map for all platforms.
